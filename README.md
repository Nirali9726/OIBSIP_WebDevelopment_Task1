# OIBSIP_WebDevelopment_Task1 — Landing Page

## Objective
To build a visually appealing and responsive landing page using HTML and CSS, demonstrating knowledge of layout design, sections, alignment, color palette, hover effects, and creative UI composition.

## Live Demo
🔗 [View Live](https://nirali9726.github.io/OIBSIP_WebDevelopment_Task1)

## Steps Performed
1. Planned the layout and sections of the landing page
2. Created the HTML structure with semantic tags — nav, section, footer
3. Built a frosted glass navbar with logo, navigation links and a Book button
4. Designed a hero section with a full screen background image and parallax scroll effect
5. Added social media icons using Font Awesome
6. Created an image card grid using CSS Grid with hover grow animations
7. Built an About section with descriptive text and a vertical city timeline
8. Added floating images using CSS absolute positioning
9. Created a What's Included section with 4 service cards
10. Added a contact form with input fields and a Send button
11. Built a footer repeating the navbar structure
12. Made the page responsive using media queries

## Tools Used
- **HTML5** — Semantic structure and layout
- **CSS3** — Flexbox, Grid, animations, transitions, gradients, backdrop-filter
- **Font Awesome** — Icons for navbar, services and social media
- **Google Fonts** — Typography
- **VS Code** — Code editor
- **GitHub Pages** — Deployment

## Outcome
A fully functional, multi-section travel landing page for Japan Tours featuring parallax scrolling, frosted glass navbar, CSS Grid image cards, vertical city timeline with floating images, contact form and footer — built using only HTML and CSS.
